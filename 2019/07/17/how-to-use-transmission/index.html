如何使用Transmission

简介

Transmission 是一个跨平台 BitTorrent 客户端，特点是一个跨平台的后端以及其上构建的简洁明了的用户界面。Transmission使用MIT和GNU双许可证授权，是一款自由软件。

Transmission 在Windows平台上的图形化用户界面使用QT构建，而在其他Unix/Linux平台上，使用更多的是它的WebUI，但是原生的WebUI简陋且缺乏语言支持，这里推荐使用第三方的transmission-web-control代替官方的WebUI。

安装

首先前往官网Transmission下载相应平台的相应版本

Windows平台

和使用msi安装普通软件无异，直接狂点下一步就行了（当然是你不介意将软件装在C盘的前提下）

Unix/Linux平台

由于我服务器使用操作系统是Ubuntu，下面的步骤仅供参考，其他类Linux系统或有不同：

使用apt安装transmission-daemon：

$ sudo apt-get install transmission-daemon

编辑配置文件：

$ vim /etc/transmission-daemon/settings.json

修改以下配置项：

需要注意的是 rpc-whitelist 项要符合自己的局域网 IP 段，如果从外网访问则需设置为外网IP 段，若不清楚也可直接将 rpc-whitelist-enabled 设置为 false 来禁用白名单。

配置保存后重载 transmission-daemon 服务：

$ sudo systemctl reload transmission-daemon.service

现在应该就可以通过浏览器访问 <服务器地址>:9091 来访问 web 控制台。

此外，推荐安装第三方的transmission-web-control代替官方的WebUI，此步骤是可选的。

已知问题

按此流程安装后会出现每次重启后无法正常连接的问题，需要重启服务：

$ sudo systemctl restart transmission-daemon.service
